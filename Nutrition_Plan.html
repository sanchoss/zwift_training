🔑 Головні правила

  1. Починай пити з перших хвилин — не чекай спраги
  2. Target: 500–750ml/h рідини indoor (менше outdoor/cool)
  3. Carbs: 30–60g/h для 1–2.5h, 60–90g/h для 2.5h+
  4. Sodium: 500–1000mg/h (Skratch дає ~380mg/scoop)
  5. Gummy Bears — залиш для перекусу після, а не під час їзди
  6. EAA — розділи: 5g до + 10g після (не під час)
  7. Мед у пляшці — game changer для indoor cycling

🔑 Key Rules

  1. Start drinking from minute 1 — don't wait for thirst
  2. Target: 500–750ml/h fluids indoor (less outdoor/cool)
  3. Carbs: 30–60g/h for 1–2.5h, 60–90g/h for 2.5h+
  4. Sodium: 500–1000mg/h (Skratch provides ~380mg/scoop)
  5. Gummy Bears — save for post-ride snack, not during
  6. EAA — split it: 5g before + 10g after (not during)
  7. Honey in the bottle — game changer for indoor cycling
